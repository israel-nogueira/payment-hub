## Como Contribuir

1. Fork o projeto
2. Crie uma branch: `git checkout -b feature/MinhaFeature`
3. Commit: `git commit -m 'Adiciona feature X'`
4. Push: `git push origin feature/MinhaFeature`
5. Abra um Pull Request

## Diretrizes

- Siga PSR-12
- Adicione testes
- Documente com PHPDoc
- Use type hints

## Testes

```bash
composer test
composer analyse
```
